# 磐石伺服器
磐石伺服器是一個邀請制的原味生存伺服器。目標是提供一個長期、穩定、不延遲的伺服器。  
目前由數位軟體工程師鎮守，務求提供穩定的服務給各位。  
預計於 **1.17** 版本開放，正在緊鑼密鼓開發中。
